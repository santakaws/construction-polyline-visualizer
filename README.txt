In Visual Studio (2022 preferably) open the CSharpPolyline solution (.sln) that is included in the CSharpPolyline folder within this package.

Once the solution/project files are loaded, you can run the program by clicking on the green play button at the top of the Visual Studio UI. You can either run it with or without debugging.

To use the app, click on the browse button and open the excel file (.xlsx) that is included in this package or any other similarly formatted excel file with different data. Filetype .xlsx is the only filetype guaranteed to work.

The app visualizes polylines which are made of interconnected line segments.
